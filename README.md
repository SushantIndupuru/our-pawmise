# Our Pawmise Website

Minimal Astro website for OurPawmise, based on the original content and expanded with multi-page operations-plan content.

## Pages

- `/` - homepage (overview, team, pets, contact)
- `/operations-plan` - overview, mission, values, and organizational context
- `/financial-assistance` - grant model details
- `/education-outreach` - preventive education and outreach model
- `/application-process` - eligibility and process details
- `/use-cases` - representative impact scenarios
- `/support` - donation page (PayPal)
- `/thank-you` - donation success page

## Development

```powershell
npm install
npm run dev
```

## Build

```powershell
npm run build
npm run preview
```
